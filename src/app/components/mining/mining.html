Conceptos Básicos

¿Qué es la minería?

Proceso mediante el cual se validan transacciones y se agregan nuevos bloques a la blockchain.

Hardware necesario

Desde CPUs hasta ASICs especializados, conoce el equipo necesario para minar.

Recompensas

Los mineros reciben bitcoins nuevos y comisiones por transacción como incentivo.

Guía práctica para comenzar

  1. Elige el hardware adecuado a tu presupuesto
  2. Descarga software de minería compatible
  3. Únete a un pool de minería o minera solo
  4. Configura tu wallet para recibir recompensas
